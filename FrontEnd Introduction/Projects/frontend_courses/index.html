html

html (Hypertext Markup Language) is the basic structure that every website follows. It defines the meaning and structure of the content on a website. Other tools are used to describe the appearence (CSS) and the function/behavior (JavaScript) of a website.

css

css (Cascading Style Sheets) is a style language (en-US) used to describe the presentation of a document written in HTML or XML (Including various XML-based languages such as SVG, MathML or XHTML). CSS defines how elements will be displayed on the screen, on paper, in speech or across other types of media.

learn more

javascript

javascript (known as JS) is an object-orientend and interpreted programming language. It is often referred to as scripting language for websites, but is also used for non-web environments such as Node.js, Apache CouchDB and Adobe Acrobat. JavaScript is a prototype-based, multi-paradigm and dynamic language, supporting object-orientated styles, imperatives and declarative (functional programming) styles.
